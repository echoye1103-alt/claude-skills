# Resume Matcher Skill

根据岗位JD（职位描述）从六个维度自动评估简历匹配度，生成结构化评分报告，辅助招聘决策或求职准备。

---

## 输入处理

### 支持的输入格式

**JD（职位描述）** 和 **简历** 均支持以下三种方式：
- 直接粘贴文本
- 上传 PDF 文件 → 使用 `pdf-reading` Skill 提取文本
- 上传 Word (.docx) 文件 → 使用 `docx` Skill 提取文本

### 输入收集流程

1. JD 和简历**均已提供** → 直接进入评估
2. **只有一项** → 礼貌询问另一项
3. 文件格式无法解析 → 请用户粘贴文本内容
4. **薪资信息缺失**（JD未写预算 或 简历未写期望）→ 该维度注明"信息不足"，不强制要求用户补充，但可提示其提供以获得更完整评估

---

## 评估维度与权重

六个维度合计满分 **100分**：

| 维度 | 满分 | 评估要点 |
|------|------|----------|
| 技能/技术栈匹配 | 25分 | 硬技能、工具、编程语言、专业技术与JD要求的重合度 |
| 项目经验 | 20分 | 项目规模、复杂度、与目标岗位的相关性、可量化成果 |
| 工作经验年限 | 20分 | 总年限是否达标、相关岗位经验、职级晋升轨迹 |
| 行业背景 | 15分 | 所在行业与目标行业的相关性、跨行业适应度 |
| 教育背景 | 12分 | 学历要求、专业方向、院校背景 |
| 薪资期望 | 8分 | 候选人薪资期望与岗位预算的匹配程度 |

---

## 评分逻辑

### 技能/技术栈匹配（0-25分）
- **21-25分**：覆盖JD要求技能80%以上，且有项目实战佐证
- **13-20分**：覆盖50%-79%，部分技能有合理替代项
- **6-12分**：覆盖30%-49%，核心技能存在明显缺口
- **0-5分**：覆盖率低于30%，严重不匹配

### 项目经验（0-20分）
- **17-20分**：有与目标岗位高度相关的项目，规模大、有量化成果（如"主导XX项目，带动收入增长30%"）
- **11-16分**：有相关项目，但规模较小或成果描述模糊
- **5-10分**：项目方向有一定相关性，但整体匹配度偏低
- **0-4分**：无相关项目经验，或项目描述过于笼统无法判断

### 工作经验年限（0-20分）
- **17-20分**：完全满足年限要求，且有直接相关岗位经验
- **10-16分**：基本满足或略低于要求，经验方向相关
- **4-9分**：经验年限不足或方向偏差较大
- **0-3分**：严重不足，几乎无相关经验

### 行业背景（0-15分）
- **13-15分**：来自同行业，行业知识直接可复用
- **8-12分**：来自上下游或强相关行业，适应成本低
- **3-7分**：跨行业，但有可迁移的通用能力
- **0-2分**：行业差异大，适应难度高

### 教育背景（0-12分）
- **10-12分**：完全符合学历和专业要求
- **6-9分**：学历达标但专业有差异，或专业符合但学历偏低一档
- **2-5分**：学历或专业有明显差距，但有其他经验补偿
- **0-1分**：学历和专业均不符合基本要求

### 薪资期望（0-8分）
- **7-8分**：期望薪资在预算范围内，且有一定谈判空间
- **4-6分**：期望薪资略高于预算（10%-20%以内），可协商
- **1-3分**：期望薪资明显超出预算（20%以上），存在较大风险
- **0分**：信息不足，无法评估（JD或简历未提供薪资信息）
- **特殊情况**：若候选人期望薪资低于预算30%以上，得满分但需在报告中注明，建议核实真实期望

---

## 综合评级

| 总分 | 评级 | 推荐行动 |
|------|------|----------|
| 85-100 | ⭐⭐⭐⭐⭐ 极度匹配 | 优先安排面试，重点候选人 |
| 70-84 | ⭐⭐⭐⭐ 高度匹配 | 建议面试，可深入了解 |
| 55-69 | ⭐⭐⭐ 中等匹配 | 可考虑面试，关注成长潜力 |
| 40-54 | ⭐⭐ 低度匹配 | 谨慎考虑，明确差距能否弥补 |
| 0-39 | ⭐ 基本不匹配 | 不建议推进，或需大量培训 |

## 使用示例

**用户输入**：
```
我有一份产品经理的JD和两份候选人简历，帮我评估一下匹配度。

【JD】
岗位：产品经理
...（JD内容）

【候选人1】
姓名：张伟
...（简历内容）
```

**Skill 输出**：
- 为每位候选人生成一份完整的评分报告
- 表格显示六维评分和综合得分
- 列举核心优势和主要差距
- 给出有针对性的建议行动

---

## 注意事项

- **客观中立**：基于事实打分，不因院校名气、姓名等主观因素加减分
- **证据引用**：优势和差距均需引用简历/JD中的具体内容，不泛泛而谈
- **薪资缺失处理**：薪资维度得0分时，在表格备注"信息不足"并在建议中提示补充
- **语言一致**：输入中文→报告中文；输入英文→报告英文；混合→默认中文
- **信息不完整**：若JD或简历关键信息缺失，报告开头注明"部分信息不足，评分仅供参考"
- **多候选人批量评估**：可同时提供多份简历与一份JD，逐一生成对比报告
